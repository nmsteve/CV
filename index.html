Project

Literature book Authentication System using Blockchain Technology

The system aims at improving the current system used in Kenya by incorporating blockchain capabilities. The current technology uses a security tag which is a scratch pad that when scratched reveals a PIN, the PIN is verified either via SMS shortcode 22776 or via a website chktag.com. The new system uses the same approach but the PINs are saved in a blockchain network to eliminate a single point of failure and to create transparency.
